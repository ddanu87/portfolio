1. semester portfolio · Multimediedesign

Mit arbejde, min proces og min læring fra 1. semester.

Dette portfolio samler mine vigtigste projekter fra semesteret og viser, hvordan jeg har arbejdet med HTML, CSS, JavaScript, UX/UI, designproces, brugertest, indhold og refleksion.

Se portfolioLæs proces

Formål

Formålet med sitet er at give et tydeligt overblik over, hvad jeg har lavet og lært på 1. semester. Sitet er bygget som en eksamensportfolio, hvor hvert projekt beskrives med opgave, proces, løsning, læring og refleksion. Fokus er ikke kun på færdige produkter, men også på de valg og metoder, der ligger bag.

Brugergrænseflader

Jeg viser arbejde med layout, navigation, visuel hierarki, typografi, farver, grid, flexbox og responsive sider.

Brugeroplevelser

Jeg dokumenterer research, målgruppeforståelse, wireframes, prototyper, test og forbedringer baseret på brugerbehov.

Frontend

Sitet og projekterne viser brug af HTML, CSS og JavaScript fra bunden samt GitHub Pages som publicering.

Udvalgte projekter

Portfolioet indeholder projekter fra grundlæggende web, UX/UI, animation/spil og virksomhedssite. Projekterne dækker forskellige dele af semesterets læringsmål og viser både teknisk arbejde, designproces og refleksion.
