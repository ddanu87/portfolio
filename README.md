# 1. semester portfolio – Multimediedesign

Dette repository indeholder mit portfolio-website til 1. semester på multimediedesigneruddannelsen.

Portfolioet samler mine projekter, min proces og min læring fra semesteret. Sitet viser arbejde med HTML, CSS, JavaScript, UX/UI, designproces, brugertest, indholdsstruktur, responsivt design, accessibility, jura, etik og refleksion.

## Struktur

- `index.html` – forside med introduktion
- `portfolio.html` – projekter fra semesteret
- `proces.html` – proces, sygdom/fravær, gruppearbejde og refleksion
- `om-mig.html` – personlig præsentation og læring
- `style.css` – styling og responsivt layout
- `script.js` – simpel interaktion/mobilmenu
- `assets/` – billeder og visuelle materialer
- `docs/` – dokumentationsfiler

## Projekter

Portfolioet indeholder projekter fra:

- Tema 2: Grundlæggende web
- Tema 3: UX/UI – OBBG.dk
- Tema 4: Animation/spil – Royal Click Fighter
- Tema 5: Virksomhedssite

Hvert projekt beskrives med opgave, proces, løsning, læring og refleksion.

## Proces og situation

Mit semester har været påvirket af sygdom, fravær og udfordringer i nogle gruppeprocesser. Derfor har portfolioet været vigtigt som en måde at samle op på mit arbejde og tydeliggøre min læring. Dette beskrives på proces-siden som refleksion og kontekst, ikke som en undskyldning.

## Teknologi

Sitet er udviklet fra bunden med HTML, CSS og JavaScript og er lavet til publicering via GitHub Pages.
